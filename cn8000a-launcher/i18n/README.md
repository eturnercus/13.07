# Локализация интерфейса

Строки интерфейса хранятся в UTF-8 JSON-файлах:

```
i18n/languages/
  ru.json   — русский (по умолчанию)
  en.json   — английский
```

## Выбор языка

1. Переменная окружения `CN8000A_LANG=ru` или `CN8000A_LANG=en`
2. Системная `LANG` (если `CN8000A_LANG` не задана)
3. Язык из сохранённого профиля (после первого подключения)
4. По умолчанию — `ru`

## Добавить свой перевод

1. Скопируйте `ru.json` в `i18n/languages/de.json` (пример для немецкого).
2. Переведите значения, **ключи не меняйте**.
3. Запустите с `CN8000A_LANG=de`.

Пример:

```json
{
  "app.title": "CN8000A KVM",
  "field.host": "KVM-Adresse",
  "button.connect": "Verbinden"
}
```

Все ключи перечислены в `ru.json` и `en.json`.
